二、VMware Workstation Pro 安装 kali Linux

2.1 创建新的虚拟机

新建一个虚拟机

2.2 添加映像文件 iso 的路径

2.3 选择客户端操作系统

Kali Linux 是基于 Debian 的 Linux 发行版, 设计用于数字取证操作系统。

2.4 配置虚拟机的名称和安装位置

输入你想要的名字，选择你需要安装的位置，下一步即可

2.5 处理器配置

按你的机器性能选择即可，然后下一步

2.6 配置虚拟机内存分配

注意：为虚拟机分配内存，建议不要超过提示的最大推荐内存，按推荐即可。

2.7 网络连接配置

提供两个链接供参考配置：
  • VMware网络连接三种配置
  • VMWare虚拟机网络连接设置

2.8 I/O 控制器类型

默认即可，直接下一步

2.9 虚拟硬盘类型

默认即可，直接下一步

2.10 配置磁盘容量分配

反正一定要比建议分配容量大就可以了，然后选择将虚拟磁盘存储为单个文件，直接下一步

2.11 新建虚拟机配置完成

点击完成，即可开始安装

三、安装 Kali Linux

3.1 开启虚拟机，进入安装界面，选择图形界面安装（Graphical install）

3.2 选择语言

选择 中文（简体）–》 中国 –》 汉语

3.3 配置系统主机名

3.4 配置用户和密码

3.5 配置磁盘分区

使用整个磁盘

将所有文件放在同一个分区中

写入磁盘

选择“是”，将改动写入磁盘中

3.6 等待系统安装完成

耐心等待即可，如果前面配置虚拟机时分配磁盘空间不足，这个时候可能会卡死，所以一定要注意前面分配足够磁盘空间

3.7 是否使用网络镜像

选否，注意此步要选No，否则一直安装不过去

3.8 将GRUB安装至硬盘

3.9 Device fro boot loader installation

一定要选择第二个 /dev/sda，因为第一个选项要 manually (手动)，不要手动，太麻烦

3.10 等待结束安装进程

3.11 安装完成，重启进入系统

四、大功告成

重启虚拟机后进入系统，输入用户名和密码，用户名默认为 root，开始使用强大的 Kali Linux 吧！
